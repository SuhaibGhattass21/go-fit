# 🛒 Go-fit

Node.js + TypeScript backend for an e-commerce platform that sells sportswear and equipment.

## ⚙️ Stack
- Node.js
- Express.js
- TypeScript
- PostgreSQL + Prisma ORM
- JWT Auth
- Swagger for API docs
- Dockerized
